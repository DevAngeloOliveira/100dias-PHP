# 100 Dias de PHP 🚀

Um desafio pessoal de 100 dias para aprofundar conhecimentos em PHP, do básico ao avançado.

## 🎯 Objetivo

Desenvolver habilidades sólidas em PHP através de exercícios práticos e projetos reais, dedicando 100 dias consecutivos ao aprendizado da linguagem.

## 🚀 Como Executar

1. Clone o repositório:
   ```bash
   git clone [URL_DO_REPOSITÓRIO]
   ```

2. Navegue até a pasta do projeto:
   ```bash
   cd 100dias-PHP
   ```

3. Inicie o servidor PHP:
   ```bash
   cd src
   php -S localhost:8000
   ```

4. Acesse no navegador:
   ```
   http://localhost:8000
   ```
